# NetPO

## Описание

NetPO - это приложение для Windows, разработанное для контроля и управления доступом к интернету и приложениям, идеально подходящее для использования на интерактивных панелях и информационных киосках. Оно позволяет пользователям устанавливать правила для разрешенных URL-адресов и приложений, а также управлять системным звуком по расписанию.

## Функции

*   **Контроль доступа к URL-адресам**: Разрешает или блокирует доступ к определенным веб-сайтам.
*   **Контроль запуска приложений**: Разрешает или блокирует запуск определенных приложений.
*   **Защита паролем**: Доступ к настройкам приложения может быть защищен паролем.
*   **Планировщик звука**: Автоматическое включение/выключение системного звука по заданному расписанию.
*   **Блогировка системных приложений**: Заблокировано сворачивание закрытие приложения, заблокировано панель управление и панель задач.

## Дополнение

Сайт по умолчанию добовляется в файл config.json в директории расположения программы. Туда же сохроняются все настройки можно после всех настроек забекапить

## Использование

1.  **Запуск приложения**: Запустите `NetPO.App.exe`.
2.  **Доступ к настройкам**: Настрйки открваются сочетанием клавиш shift + f1
3.  **Настройка URL-адресов**: В разделе "URL" вы можете добавлять разрешенные URL-адреса.
4.  **Настройка приложений**: В разделе "Приложения" вы можете добавлять разрешенные исполняемые файлы (например, `notepad.exe`).
5.  **Настройка расписания звука**: В разделе "Расписание" вы можете добавить время в формате HH:mm, когда системный звук будет переключаться (включаться/выключаться).
6.  **Сохранение настроек**: После внесения изменений нажмите "Сохранить и закрыть".
